# Portfolio

The live site can be found [here](https://nickjmorrow.com/).

## Technical Stack: Gatsby, NodeJS, PostgreSQL

-   The **front-end** is built using React with a heavily reliance on a [personal component library](https://nickjmorrow.github.io/react-component-library/).
-   The **back-end** is built using NodeJS wrapped in GraphQL. I wanted an easy, light-weight backend and to be able to easily specify the data shape without doing much extra work.
-   The **database** is PostgreSQL. I wanted something relational, given the data is heavily relational. No strong reasons for choosing PostgreSQL other than it being the _hot new thing_.
